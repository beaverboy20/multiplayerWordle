# Multiplayer Wordle

a multiplayer version of wordle — no server required. 
connect directly to each other via a peer-to-peer connection and guess the word together.

## How to Play

1. both players have to be on the same network. for seperate networks, use a free unneling software. https://www.radmin-vpn.com/ 
2. one player hosts a game and shares their connection code with a friend.
3. the other player enters the code to join.
4. both players guess the same 5-letter word simultaneously.
